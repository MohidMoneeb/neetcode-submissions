# NeetCode — solutions + my notes

Working through NeetCode. My raw solutions sync automatically from NeetCode.io into
`Data Structures & Algorithms/`. What I actually care about is the **notes**: for each
problem I write down how I thought about it, what I tried first, what I'd change, and a
short blurb I can reread before an interview.

The point isn't to have "correct answers" sitting in a folder. It's to be able to
re-derive them under pressure, and to show my thinking, not just my output.
